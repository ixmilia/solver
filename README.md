# solver

An AI-assisted TypeScript port of the [ISOtope](https://github.com/CADmium-Co/ISOtope) geometric constraint solver.

## Building

```sh
npm install
npm run build
```

## Testing

```sh
npm test
```

## Bundling for the browser

```sh
npm run publish
```

This produces a self-contained `www/` directory with `index.html` and `solver.bundle.js`. Serve it locally with:

```sh
npm run serve
```
